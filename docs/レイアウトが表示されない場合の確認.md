# 詳細タブに項目が表示されない場合の確認（レコードタイプ別レイアウト割り当て）

「Question Layout」を割り当て済みでも、起票者・起票日時・内容などが詳細に表示されない場合は、**レコードタイプごとにレイアウトが割り当てられている**可能性があります。

---

## 手順：レコードタイプ別に「Question Layout」を割り当てる

1. **設定** → **オブジェクトマネージャ** → **問い合わせ** をクリック

2. 左メニューで **ページレイアウト** をクリック

3. **割り当て**（または **レイアウトの割り当て**）をクリック

4. 表示された表を確認します。
   - 左列：**プロファイル名**
   - 右側：**レコードタイプごとの列**（「マスター」「LWCテスト」など）

5. **あなたがログインしているユーザーのプロファイル**の行で、**「LWCテスト」の列**（表示されているレコードタイプの列）をクリックし、ドロップダウンから **Question Layout** を選択する  
   - 「マスター」の列も同様に **Question Layout** にしておくと安心です。

6. **保存** をクリック

7. 問い合わせレコードの詳細ページを**再読み込み（F5）**して、起票者・起票日時・対応期日・優先度・対象顧客・内容が表示されるか確認する

---

## 補足

- レコードタイプが1つもない場合は、表には「マスター」だけが表示されます。その場合は「マスター」の列で **Question Layout** が選ばれているか確認してください。
- 「Question Layout」を開いて編集し、**起票者**・**起票日時**・**内容**などの項目がレイアウトに含まれているかもあわせて確認するとよいです。
